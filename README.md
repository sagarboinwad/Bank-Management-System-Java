# Bank Management System (Java Console Application)

A simple **Bank Management System** developed using **Core Java** concepts like **Inheritance**, **Encapsulation**, and **Polymorphism**.  
The system allows users to create Savings and Current accounts, deposit/withdraw money, and view account details.

---

## Technologies Used
- Java (Core)
- OOP Concepts (Inheritance, Encapsulation, Polymorphism)
- ArrayList (Java Collections)
- Console-based UI

---

## Key Features
- Create **Savings** and **Current** Accounts
- Deposit and Withdraw money
- Add interest (Savings accounts only)
- Display all or specific account types
- Use of `ArrayList` to manage multiple accounts
- Menu-driven interactive console interface

---

## Object-Oriented Concepts Demonstrated
| Concept | Implementation |
|----------|----------------|
| **Encapsulation** | Private fields with getters and setters |
| **Inheritance** | `SavingsAccount` and `CurrentAccount` extend `Account` |
| **Polymorphism** | Dynamic method calls through `ArrayList<Account>` |
| **Abstraction (optional)** | Base class `Account` as abstract |
